About FontEnd Languages

Le développement web frontal (aussi appelé front-end en anglais) correspond aux productions HTML, CSS et JavaScript d’une page internet ou d’une application qu’un utilisateur peut voir et avec lesquelles il peut interagir directement.
  • HTML décrit le contenu et définit la structure dans une page Web
  • CSS est un ensemble de règles qui décrivent la priorité avec laquelle les styles sont rendus sur une page
  • JavaScript sert à contrôler les données saisies dans des formulaires HTML, ou à interagir avec le document HTML via l'interface Document Object Model ou DOM, fournie par le navigateur

HTML

Hyper Text Markup Language
HTML5, est le langage de balisage conçu pour représenter les pages web. C’est un langage permettant d’écrire de l’hypertexte, d’où son nom. HTML permet également de structurer sémantiquement et logiquement et de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des formulaires de saisie et des programmes informatiques.
  • Développé par : World Wide Web Consortium & WHATWG
  • Version initiale : 1992
  • Dernière version : HTML5

CSS

Cascading Style Sheets
Les feuilles de style en cascade1, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML. Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.
  • Développé par : CSS Working Group
  • Version initiale : 17 décembre 1996
  • Dernière version : CSS3

JS

Java Script
JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs2 avec l'utilisation (par exemple) de Node.js3. C'est un langage orienté objet à prototype, c'est-à-dire que les bases du langage et ses principales interfaces sont fournies par des objets qui ne sont pas des instances de classes, mais qui sont chacun équipés de constructeurs permettant de créer leurs propriétés.
  • Développé par : Netscape Communications Corporation
  • Version initiale : 4 décembre 1995
  • Dernière version : JS 8
